# GameState Model

This is the core data structure shared between phone and watch.

## Fields

- inning (Int)
- isTop (Boolean)
- balls (Int)
- strikes (Int)
- outs (Int)
- homeScore (Int)
- awayScore (Int)
- lastUpdated (Timestamp)

## Example

Top 4th inning  
Count: 2-1  
Outs: 1  
Score: Home 4, Away 3
